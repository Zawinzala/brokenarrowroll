## 贡献指南

我们欢迎对项目的贡献！如果您想要改动，请遵循以下原则：

### 词条修改原则

1. ：不要删除某个贡献过的作者
2. ：不要大量删除某个作者的词条
3. ：删除词条之前尽可能尝试优化而非删除
5. ：大量变动可以备注更新的内容

### 贡献流程

#### 方式1. Fork 本仓库 并修改 pull
#### 方式2. 直接朝我要合作权限，并遵守词条修改原则
